# SSH-keeper
Easy access to SSH connections.


##### INSTALL
* `cd /tmp`
* `git clone https://github.com/klem4/SSH-keeper.git`
* `cd SSH-keeper/`
* `sudo make`

Then just type "s" for start using program.

##### UNINSTALL
* `sudo make clean`
